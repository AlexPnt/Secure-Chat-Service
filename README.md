Programming and Security Assignment
===================================
Secure-Chat-Service
-------------

#### Features: ####
* Multiclient Support
* Confidentiality
* Authenticity
* Integrity
* Non-repudiation
* Key management

####Installation:####
In the root directory of the project type:
    
    mkdir bin
    javac src/*.java -d bin/

####Usage:####
**Running the server**:

    cd bin/
    java ChatServer <hostname> <port>
    
or simply

    cd bin/
    java ChatServer

**Running the client**:

    cd bin/
    java ChatClient <hostname> <port>
    
or simply

    cd bin/
    java ChatClient


Cheers!
